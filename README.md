# Тестовое задание

Запустить: docker-compose up -d

Автоматически сгенерированная документация: http://localhost:8001/docs 



GET: получить список имен таблиц, которые уже находятся в базе данных

POST: обработать новый csv или xlsx файл

  -обязательный параметр: входной файл в формате xlsx или csv, название файла станет названием таблицы
  
  -необязательные параметры:
  
    -header(число) - номер строки, которая будет содержать названия колонок, с нее начинают считываться данные, по умолчанию 0(нумерация с 0)
    
    -names - список названий колонок(количество введенных названий должно соответствовать числу колонок), по умолчанию в качестве названий буду использованы названия колонок из строки header
    
    -index_col -  список названий колонок, на которые необходимо создать индексы (должны соответствовать существующим колонкам) - по умолчанию не задано: в таком случае будет создана колонка 'id'.
    
    

В файле tests.py находятся тесты(точнее один тест для get, т.к. у мне к сожалению не удалось справиться с тестированием uploadfile), можно запустить с помощью pytest.

Дополнительные комментарии по реализации можно найти в коде.
